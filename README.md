# Pulsetech — Landing Page

Sitio web institucional desarrollado para Pulsetech, una agencia de desarrollo web orientada a negocios que necesitan presencia digital profesional.

---

## Sobre el proyecto

El encargo fue construir la landing page de la propia agencia: el sitio que un potencial cliente ve antes de decidir si contrata o no. Eso implicó un nivel de exigencia alto en diseño, performance y coherencia visual, ya que el sitio en sí mismo es la carta de presentación del trabajo que la agencia ofrece.

Todo el diseño y desarrollo fue realizado de forma autónoma, desde la arquitectura del proyecto hasta las decisiones visuales y de contenido.

---

## Stack

Desarrollada con **Astro**, por su modelo de generación estática que garantiza tiempos de carga mínimos y métricas de Core Web Vitals sobresalientes — especialmente relevante para un sitio cuyo argumento de venta es, entre otras cosas, la performance web.

---

## Desafíos técnicos

**Coherencia entre discurso y ejecución:** La agencia promete sitios rápidos, bien construidos y sin shortcuts. El propio sitio tenía que demostrarlo. Cada decisión técnica fue tomada con ese criterio: cero dependencias innecesarias, CSS propio sin frameworks de UI, imágenes optimizadas y estructura de componentes limpia.

**Diseño sin referencias directas:** Al ser el sitio de la agencia y no de un cliente, no había un brief externo ni una identidad preexistente. Las decisiones de paleta, tipografía y layout fueron completamente propias, lo que requirió más iteración que un proyecto con lineamientos definidos.

**Contenido y arquitectura de información:** Comunicar servicios, diferencial y llamado a la acción en una sola página sin que se sienta saturado fue uno de los principales desafíos no técnicos del proyecto.
